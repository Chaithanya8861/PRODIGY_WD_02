## 📌 Overview
Premium watch-themed stopwatch with lap tracking, keyboard shortcuts, and glass-morphism design.

## ✨ Features
- ⏯️ Start / Pause / Reset
- 🏁 Record unlimited laps
- 🗑️ Clear all laps
- ⌨️ Keyboard: `Space` (Start/Pause), `R` (Reset), `L` (Lap), `C` (Clear)
- 📱 Fully responsive
